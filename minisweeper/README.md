# Minisweeper Project

這是一個使用 HTML, CSS, 和 JavaScript 構建的經典踩地雷遊戲 (Minesweeper)。
包含了一個**自動解題機器人 (Auto Solver)**，它會模仿人類的思考方式來幫你玩。

## 🤖 機器人是怎麼思考的？ (簡單版)

這個機器人並不「作弊」（它不知道地雷藏在哪），它只是算術很快。它只用兩招小學數學等級的邏輯：

### 第一招：空位剛好夠放雷 (定雷)
**「如果我要找 3 顆雷，而周圍剛好只剩 3 個洞，那這些洞肯定全都是雷！」**

*   **情境**: 一個格子顯示 `3`。
*   **觀察**: 周圍未打開的格子剛好是 `3` 個。
*   **結論**: 這 3 個格子全部標記為地雷🚩。

### 第二招：雷都找齊了 (排雷)
**「如果我要找 2 顆雷，而周圍已經插了 2 支旗子，那剩下的格子肯定都安全！」**

*   **情境**: 一個格子顯示 `2`。
*   **觀察**: 周圍已經有 `2` 個旗子🚩。
*   **結論**: 周圍剩下的未知格子全部點開😊，絕對安全。

---

### 機器人的工作流程

1.  **掃描全場**: 它會盯著版面上的每一個數字看。
2.  **判斷**: 對每個數字問：「可以用第一招嗎？」或「可以用第二招嗎？」
3.  **執行**: 如果可以，它就自動幫你插旗或點開（每一步會停頓 0.1 秒讓你欣賞）。
4.  **重複**: 因為點開了新格子，會有新數字出現，它就繼續重複上述步驟。
5.  **放棄**: 如果掃完整場，發現沒有任何格子可以用這兩招解決（例如遇到必須二選一猜運氣的情況），它就會停下來交給你。
